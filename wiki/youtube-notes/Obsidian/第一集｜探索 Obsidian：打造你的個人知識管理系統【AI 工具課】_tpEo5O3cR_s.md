# [基礎入門] 探索 Obsidian：打造個人知識管理系統

### 💡 一句話核心精華
Obsidian 是一個以「知識點串聯」為核心、檔案在地化存儲的強大筆記工具，透過 Markdown 格式實現高度靈活的個人知識管理。

---

### 📥 軟體概覽與安裝 (00:00)
*   **軟體屬性**：Obsidian 是一套個人知識管理系統（PKM），採「在地端（Local-first）」運行，資料儲存在使用者自己的電腦中。
*   **同步機制**：若需雲端同步，可購買官方提供的專用帳號授權，或手動備份檔案。
*   **知識邏輯**：強調「知識點與知識點的串聯」，與傳統長篇大論的筆記軟體不同。
*   **安裝流程**：
    1. 前往官網下載對應版本（支援 Mac、Windows）。
    2. 安裝後開啟，於右下角設定中將語系更改為「繁體中文」。

### 📂 建立儲存庫 Vault (02:15)
*   **核心概念**：Obsidian 的「儲存庫（Vault）」本質上就是電腦裡的一個「資料夾」。
*   **建立步驟**：
    1. 點擊「建立」新儲存庫。
    2. 命名儲存庫（如：課程資料）。
    3. 選擇存放位置（如：桌面或特定雲端硬碟資料夾）。

### 🖥️ 介面配置與操作 (03:40)
*   **三大區塊**：
    1. **左側工具欄**：檔案總管、搜尋、新增筆記等功能。
    2. **中間編輯區**：撰寫內容的地方，支援「分割視窗」（左、右視窗），方便對照資料。
    3. **標籤頁管理**：右鍵點擊標籤頁可進行關閉或分割。
*   **筆記管理**：
    *   **新增**：點擊「新增筆記」圖示。
    *   **刪除**：在左側筆記名稱上點擊右鍵選擇「刪除」。

### 📝 筆記撰寫策略：原子化筆記 (05:40)
*   **單一資訊原則**：Obsidian 不建議寫成超長頁面。建議將資料切碎為「單一知識點」（如：一個定義、一個想法）。
*   **檔案格式**：每一頁筆記本質上都是一個 `.md` (Markdown) 檔案。

### 🔄 檔案系統同步邏輯 (07:20)
*   **雙向連動**：Obsidian 的側邊欄結構與電腦檔案總管的資料夾結構是完全一致且同步的。
    *   在電腦資料夾中新增子資料夾，Obsidian 會立刻顯示。
    *   在 Obsidian 中移動檔案，電腦實體資料夾也會跟著移動。
*   **優勢：開放與移植性**：
    *   檔案不被軟體綁架，即使不開啟 Obsidian，也能用任何純文字編輯器開啟 `.md` 檔。
    *   分享資料非常簡單，只需複製實體 MD 檔案或資料夾即可。

### 🛠️ 快速操作重點
| 功能 | 說明 |
| :--- | :--- |
| **語系設定** | 右下角設定 -> Language -> 繁體中文 |
| **儲存庫 (Vault)** | 筆記所在的實體資料夾 |
| **Markdown (.md)** | Obsidian 筆記的儲存格式 |
| **視窗分割** | 標籤頁右鍵 -> 往右/下分割視窗 |
| **資料夾管理** | 支援多層級資料夾，與作業系統同步 |
