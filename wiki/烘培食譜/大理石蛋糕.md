# 大理石蛋糕 (Marble Cake)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (應檢重量：製作麵糊重 500g/個，長條形蛋糕 4/5/6 個)
- 耗損 10%
- 白麵糊與黑麵糊比例約為 5:1

### 白麵糊 (4個)
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 酥油 | 45 | 258 |
| 白油 | 45 | 258 |
| 低粉 | 100 | 574 |
| 泡打粉 | 1.6 | 9 |
| 鹽 | 1 | 6 |
| 糖粉 | 92 | 528 |
| 蛋 | 83 | 476 |
| 奶水 | 16 | 92 |
| **合計** | **383.6** | **2199** |

### 黑麵糊
| 材料 | 百分比 (%) | 重量 (g) | 備註 |
| :--- | :--- | :--- | :--- |
| 白麵糊 | 100 | 305 | 取白麵糊約 1/6 量 |
| 熱水 | 13 | 40 | |
| 可可粉 | 6.5 | 20 | |
| 小蘇打 | 0.5 | 1 | |
| **合計** | **120** | **366** | 白麵糊 1/6 量 |

## 製作步驟

### 準備工作：
1. 折好長型烤模用的烤焙紙，放入模中備用。

### 白麵糊：
1. **酥油、白油、低粉、泡打粉**放入攪拌缸，打至絨毛狀。
2. 加入**糖粉、鹽**慢速攪拌均勻。
3. **蛋**液分次加入，中速攪拌至比重約 0.75~0.8。
4. 加入**奶水**攪拌均勻。

### 黑麵糊：
1. 取**熱水、小蘇打粉、可可粉**混合拌勻。
2. 加入約 1/6 的**白麵糊**混合拌勻即為黑麵糊。

### 組合與烘理：
1. 將黑麵糊加入白麵糊中，隨意撥動約 3 次產生大理石紋路。
2. 填入長型烤模，輕敲去氣泡。
3. 入爐烤培。**上火 200°C / 下火 170°C**，烤 15 分鐘使皮結皮。
4. 出爐用刀在表面劃出直線出線（利於龜裂美觀），降溫續烤。
5. **上火 180°C / 下火 170°C**，再烤約 20 分鐘轉向再烤 20 分鐘。
6. 出爐後重敲，脫膜散熱放涼。
