# 泡芙 (奶油空心餅 / Cream Puffs)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (實作重量：麵糊重 650/700/800 克)
- 耗損 10%
- 製作 16/18/20 個，烤好後取 10 個切開填入 50g 奶油布丁餡。

### 泡芙麵糊 (650g)
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 奶油 | 30 | 45 |
| 沙拉油 | 70 | 106 |
| 水 | 100 | 151 |
| 鹽 | 2 | 3 |
| 高筋麵粉 | 100 | 151 |
| 全蛋 | 170 | 257 |
| **合計** | **472** | **714** |

### 奶油布丁餡 (550g)
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 全蛋 | 16 | 61 |
| 細砂糖 | 20 | 77 |
| 玉米粉 | 6 | 23 |
| 低筋麵粉 | 6 | 23 |
| 鮮奶 | 100 | 382 |
| 奶油 | 10 | 38 |
| **合計** | **158** | **605** |

## 製作步驟

### 麵糊：
1. 將**奶油、沙拉油、水、鹽**煮至沸騰。
2. 加入**高筋麵粉**快速拌煮直到麵粉完全糊化。
3. 麵糊移入攪拌缸，**蛋液**分 3 次加入。每倒完一次蛋液後，中速攪拌並每隔 2 分鐘重啟攪拌一次（共 3 次），直到麵糊提起來呈倒三角形緩慢掉落狀態。
4. 麵糊裝入擠花袋，使用平口花嘴，將麵糊平均擠在烤盤上。
5. 入爐前表面噴水噴霧。
6. 入爐烤培。**上火 190°C / 下火 170°C**，烤 20min + 15min。

### 奶油布丁餡：
1. **低筋麵粉、玉米粉、細砂糖、蛋液**一起拌勻。
2. 將煮沸的**鮮奶**倒入(1)中快速拌勻。
3. 再煮沸至呈稠狀後關火，最後加入**奶油**拌勻。
4. 待涼後備用。

### 組合：
1. 取 10 個泡芙從中間切開（不切斷），填入約 50g 奶油布丁餡即可。
