# 圓頂葡萄乾土司 (Raisin Round Top Bread)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (製作 4 條，每條麵糰重 560g)
- 耗損 5%

| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 高筋麵粉 | 100.0 | - |
| 細砂糖 | 15.5 | - |
| 鹽 | 1.2 | - |
| 奶粉 | 3.5 | - |
| 水 | 54.0 | - |
| 全蛋 | 10.0 | - |
| 酵母 | 2.0 | - |
| 奶油 | 10.0 | - |
| 葡萄乾 | 20/25/30 | - |
| **合計** | **216.2 / 221.2 / 226.2** | - |

## 製作步驟

1. **準備**：葡萄乾先泡水（軟化後瀝乾）。
2. **攪拌**：除奶油外的所有材料放入攪拌缸。慢速 1 分鐘，中速 10 分鐘，攪拌至擴展階段。
3. **加入油脂**：加入**奶油**，中速攪拌約 3 分鐘至完成階段。
4. **加入葡萄乾**：將瀝乾的葡萄乾加入麵糰，慢速攪拌 1 分鐘使分佈均勻。
5. **基本發酵**：麵糰滾圓，進行基本發酵 50 分鐘。
6. **分割與中間發酵**：分割麵糰，每顆 560g。滾圓後中間發酵 15 分鐘。
7. **整形**：整形後捲入模中。確保葡萄乾儘量不要外露於表面以防烤焦。
8. **最後發酵**：最後發酵 40~60 分鐘，發至模高約八分滿。
9. **烘烤**：入爐烘烤。**上火 150°C / 下火 200°C**，烤約 20 分鐘轉向再烤 20 分鐘。
