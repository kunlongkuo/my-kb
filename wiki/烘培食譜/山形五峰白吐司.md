# 山形五峰白吐司 (Mountain White Bread)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (製作 3 條，每條 900g，每條由 5 顆 180g 麵糰組成)
- 耗損 5%

| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 高筋麵粉 | 100.0 | - |
| 細砂糖 | 5 / 7 / 9 | - |
| 鹽 | 1.7 | - |
| 奶粉 | 4.0 | - |
| 水 | 64.0 | - |
| 酵母 | 2.0 | - |
| 白油 | 5 / 7 / 9 | - |
| **合計** | **181.7 / 185.7 / 189.7** | - |

## 製作步驟

1. **攪拌**：除油脂外的所有材料放入攪拌缸。慢速 1 分鐘，中速 10 分鐘，攪拌至擴展階段。
2. **加入油脂**：加入**白油**，中速攪拌約 3 分鐘至完成階段（撐開呈薄膜狀）。
3. **基本發酵**：麵糰滾圓，進行基本發酵 20 分鐘（環境 28°C，濕度 75%）。
4. **分割與中間發酵**：分割麵糰，每顆 180g（一條吐司需 5 顆）。滾圓後中間發酵 5 分鐘。
5. **整形與發酵**：
    - **第一次桿捲**：桿開後折三折。中間發酵 10~15 分鐘。
    - **第二次桿捲**：桿長後捲起。
6. **入模**：放入吐司模。入模順序建議為：4-2-1-3-5。
7. **最後發酵**：進入發酵箱進行最後發酵 40~60 分鐘（環境 38°C，濕度 85%），發至模高約 9.5 分滿。
8. **烘烤**：入爐烘烤。**上火 150°C / 下火 200°C**，烤約 25 分鐘轉向再烤 25 分鐘。
