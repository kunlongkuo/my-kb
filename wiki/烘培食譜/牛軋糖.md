## 製作方式分類
牛軋糖主要依據「蛋白質來源」與「糖漿處理」分為兩大類：
1. **熬糖版 (正統式)**: 使用麥芽糖、細砂糖、鹽及水熬煮至 135℃-140℃，再衝入打發的蛋白霜（或蛋白粉液）中。口感較紮實、香氣足。
2. **棉花糖版 (簡易式)**: 利用棉花糖已含有的明膠與蛋白成分，加熱融化後加入奶粉與奶油拌勻。適合家庭快速製作。

## 典型配方 (參考)
| 原料名稱 | 功能 |
| :--- | :--- |
| 水飴 / 麥芽糖 | 主體結構，控制軟硬度 |
| 海藻糖 / 細砂糖 | 甜度來源 (海藻糖甜度較低) |
| 蛋白霜 / 蛋白粉 | 賦予蓬鬆空氣感與白色外觀 |
| 奶油 (無鹽) | 增加奶香與潤滑感 |
| 全脂奶粉 | 乳香來源與硬度支撐 |
| 堅果 (杏仁/花生) | 增加口感與香氣 (需事先低溫烘烤保溫) |

## 關鍵技術節點 (根據助教筆記)
- **溫度控制**: 糖漿熬煮溫度是決定牛軋糖硬度的關鍵（夏天建議高 2-3 度，冬天可低 2-3 度）。
- **保溫堅果**: 拌入的堅果必須是溫熱的（約 60-80℃），否則會導致糖漿迅速硬化，難以拌勻及整形。
- **成型**: 趁熱倒入模具，壓平，待核心溫度降至室溫但尚未全硬時切割最為整齊。
