# 圓頂奶油吐司 (Cream Round Top Bread)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (製作 3/4/5 條，每條麵糰重 560g)
- 耗損 5%

| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 高筋麵粉 | 100.0 | - |
| 細砂糖 | 10.0 | - |
| 鹽 | 1.0 | - |
| 奶粉 | 3.5 | - |
| 水 | 52.0 | - |
| 全蛋 | 12.0 | - |
| 酵母 | 1.2 | - |
| 奶油 | 10.0 | - |
| **合計** | **189.7** | - |

## 製作步驟

1. **攪拌**：除奶油外的所有材料放入攪拌缸。低速 1 分鐘，中速 10 分鐘，攪拌至擴展階段。
2. **加入油脂**：加入**奶油**，中速攪拌約 3 分鐘至完成階段。
3. **基本發酵**：麵糰滾圓，進行基本發酵 50 分鐘。
4. **分割與中間發酵**：分割麵糰，每顆 560g。滾圓後中間發酵 15 分鐘。
5. **整形**：整形後捲入模中。
6. **最後發酵**：最後發酵 40~60 分鐘，發至模高約七分滿。
7. **烘烤**：入爐烘烤。**上火 150°C / 下火 200°C**，烤約 20 分鐘轉向再烤 20 分鐘。
