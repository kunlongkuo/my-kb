# 巧克力戚風蛋糕捲 (Chocolate Chiffon Cake Roll)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (應檢重量：麵糊重 1800g/1900g/2000g 一盤)
- 耗損 10%
- 奶油霜 300g (考場提供)

| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 可可粉 | 22 | 66 |
| 小蘇打 | 1 | 3 |
| 熱水 | 80 | 241 |
| 沙拉油 | 74 | 223 |
| 細砂糖 (1) | 56 | 169 |
| 蛋黃 | 74 | 223 |
| 低筋麵粉 | 100 | 301 |
| 蛋白 | 148 | 445 |
| 塔塔粉 | 0.3 | 1 |
| 鹽 | 0.6 | 2 |
| 細砂糖 (2) | 100 | 301 |
| **合計** | **655.9** | **1974** |

## 製作步驟

### 巧克力麵糊：
1. 將**可可粉、小蘇打、熱水**一起拌勻。
2. 加入**沙拉油、細砂糖(1)**拌勻。
3. 再加入**蛋黃**一起拌勻。
4. **低筋麵粉**與過篩後，分次加入拌勻至無粉狀之巧克力麵糊。

### 蛋白霜：
1. **蛋白、塔塔粉、鹽**放入缸中，以中速打發。
2. **細砂糖(2)**分兩次加入，打發至濕性發泡（比重約 0.16~0.18）。

### 組合與烘理：
1. 取 1/3 蛋白霜加入巧克力麵糊中稍拌後，再將剩餘蛋白霜一起加入拌勻。
2. 將拌好的巧克力蛋糕糊倒入已經鋪好白紙的烤盤中整平。
3. 入爐烤培。**上火 185°C / 下火 150°C**，烤約 20min + 10min。
4. 蛋糕出爐後重敲，放置於涼架撕開四邊白紙散熱放涼。
5. 將蛋糕翻面放置於新烤焙紙上，撕去底部烤焙紙，抹上奶油霜後捲成蛋糕捲。
6. 切成 30cm 蛋糕條二條。
