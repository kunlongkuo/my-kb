# 海綿蛋糕 (Sponge Cake)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (應檢重量：製作麵糊重 550g/個，8吋 3/4/5 個)
- 耗損 10%

| 材料 | 百分比 (%) | 重量 (g) (3個量) |
| :--- | :--- | :--- |
| 全蛋 | 185 | 779 |
| 蛋黃 | 14 | 59 |
| 細砂糖 | 100 | 421 |
| 鹽 | 1 | 4 |
| 低筋麵粉 | 100 | 421 |
| 香草粉 | 1 | 4 |
| 沙拉油 | 15 | 63 |
| 奶水 | 15 | 63 |
| **合計** | **431** | **1815** |

## 製作步驟

1. **打發**：將**全蛋、蛋黃**隔水加熱至約 40~43°C，使用球形攪拌器快速攪拌約 3 分鐘。
2. 加入**細砂糖、鹽**，繼續攪拌 2~3 分鐘至蛋液呈乳白色且濃稠。
3. **拌粉**：慢速分次加入過篩後的**低筋麵粉、香草粉**，輕輕拌勻。
4. **拌油水**：最後加入**沙拉油、奶水**拌勻。
5. **烘理**：將麵糊倒入烤模（每個約 550g）。
6. 入爐烤培。**上火 180°C / 下火 180°C**，烤約 20min + 15min。
7. **出爐**：出爐後重敲並倒扣散熱，待完全涼透後脫模即可。
