# 紅豆甜麵包 (Red Bean Sweet Bread)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (製作 18/20/22 顆，每顆麵糰 60g，紅豆餡 30g)
- 耗損 5%

### 麵糰材料
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 高筋麵粉 | 100.0 | - |
| 細砂糖 | 19.0 | - |
| 鹽 | 1.0 | - |
| 奶粉 | 3.5 | - |
| 水 | 54.0 | - |
| 全蛋 | 10.0 | - |
| 酵母 | 2.0 | - |
| 奶油 | 9.0 | - |
| **合計** | **198.5** | - |

### 內餡材料
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 紅豆餡 | 100.0 | - |

## 製作步驟

1. **攪拌**：除油脂外的所有材料放入攪拌缸。慢速 1 分鐘，中速 10 分鐘。加入**奶油**後中速約 3 分鐘至完成階段。
2. **基本發酵**：麵糰滾圓，基本發酵 40 分鐘。
3. **分割與中間發酵**：分割麵糰，每顆 60g（18/20/22 顆）。滾圓後中間發酵 15 分鐘。
4. **包餡**：將紅豆餡秤重（每顆 30g），包入麵糰內並整成圓形。
5. **最後發酵**：進行最後發酵 40~60 分鐘，至麵糰直徑約 8.5cm。
6. **烘烤**：入爐烘烤。**上火 195°C / 下火 185°C**，烤約 8 分鐘轉向再烤 4 分鐘即可。
