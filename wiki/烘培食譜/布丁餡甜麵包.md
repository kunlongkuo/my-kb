# 布丁餡甜麵包 (Pudding Sweet Bread)

本食譜摘自丙級烘焙技術士筆記與指定參考配方。

## 配方 (製作 18/20/22 顆，每顆麵糰 60g，布丁餡 30g)
- 耗損 5%

### 麵糰材料
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 高筋麵粉 | 100.0 | - |
| 細砂糖 | 19.0 | - |
| 鹽 | 1.0 | - |
| 奶粉 | 3.5 | - |
| 水 | 54.0 | - |
| 全蛋 | 10.0 | - |
| 酵母 | 2.0 | - |
| 奶油 | 9.0 | - |
| **合計** | **198.5** | - |

### 布丁內餡材料
| 材料 | 百分比 (%) | 重量 (g) |
| :--- | :--- | :--- |
| 全蛋 | 16.0 | - |
| 細砂糖 | 20.0 | - |
| 玉米粉 | 6.0 | - |
| 低筋麵粉 | 6.0 | - |
| 鮮奶 | 100.0 | - |

## 製作步驟

### 麵糰：
1. **攪拌**：除奶油外的所有材料放入攪拌缸。低速 1 分鐘，中速 10 分鐘。加入**奶油**後中速約 3 分鐘至完成階段。
2. **基本發酵**：麵糰滾圓，基本發酵 50 分鐘（環境 28°C，濕度 75%）。

### 布丁餡：
1. **細砂糖 + 蛋液**拌勻，加入過篩的**玉米粉、低筋麵粉**一起拌勻。
2. 將煮滾的**奶水**沖入(1)中，快速攪拌並煮至濃稠狀後，加入**奶油**，放涼即可。

### 組合與烘烤：
1. 分割麵糰，每顆 60g（18/20/22 顆）。滾圓後中間發酵 15 分鐘。
2. 將布丁餡秤重（每顆 30g），包入麵糰內並整成圓形。
3. **最後發酵**：最後發酵 40~60 分鐘（環境 38°C，濕度 85%），至麵糰直徑約 8.5cm。
4. **烘烤**：入爐烘烤。**上火 195°C / 下火 185°C**，烤約 8 分鐘轉向再烤 4 分鐘即可。
