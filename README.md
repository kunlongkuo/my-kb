# Mark's Personal Knowledge Base (my-kb) 🧠✨

Welcome to my personal Second Brain! This repository is a highly structured, systemic knowledge vault designed for personal learning, reference, and AI-assisted retrieval. 

It is built as an **Obsidian Vault** and optimized for **LLM Wiki / AI Copilot** integration under the **Anti-Gravity** framework.

---

## 📂 Vault Structure

This knowledge base is divided into structured folders according to the [Knowledge Base Schema](schema.md):

*   **`wiki/`**: Curated, high-quality knowledge nodes organized by category:
    *   `交通法規/` - Traffic regulations & penalty guidelines
    *   `勞工法規/` - Labor laws, pension systems, and leaves
    *   `金融投資/` - Market-wide ETF lists (dividend, leverage, active, etc.)
    *   `烘培食譜/` - Professional baking recipes and technician exam guidelines (Cakes & Breads)
    *   `AI工具/` - Workflows for LLMs, script automation, and AI image extraction
    *   `System/` - Core wiki guidelines and system metadata
*   **`templates/`**: Obsidian templates for quick page creation
*   **`raw/`**: Source text transcripts, drafts, and raw assets
*   **`cards/`**: Atomic idea cards awaiting synthesis
*   **`scripts/`**: Automation scripts (e.g. YouTube summarizer, data pipelines)
*   **`chrome_extension/`**: Custom browser tools for clipping and exporting data
*   **`gemini-kb-app/`**: Workspace-local applications and integrations
