## 🚀 如何在 Colab 實現每日自動化執行？
1. **使用 Colab Pro 排程器**：
   - 點擊畫面右上方「三角形箭頭 (執行時間)」旁邊的「排程筆記本」。
   - 在彈出視窗中設定「每日 (Daily)」執行時間。
   - 確保您的 API Key 與密碼已存放在左側「🔑 (Secrets)」中，並使用 `userdata.get()` 讀取。
2. **保持環境持久性**：
   - 將爬取後的資料定期存入「Google Drive」，避免 Runtime 關閉後資料遺失。
   - 使用「Gemini 1.5 Flash」以確保在大規模自動化時保有速度與成本優勢。
3. **錯誤警示**：
   - 在主函數中加入 `try-except`，若發生錯誤可透過 Line 發送「異常告警」。
