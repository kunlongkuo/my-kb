# 資訊圖表生成（Infographic-Builder）

本技能旨在協助使用者根據不同的展示需求，設計並生成高品質的資訊圖表（如流程圖、關係圖、統計圖或海報插圖），並自動建議最適用的渲染技術。

## 執行流程 (SOP)

當載入此技能時，請依照以下步驟引導使用者：

### STEP 1：釐清圖表需求與類型
向使用者確認以下重點，以決定最適合的製作技術：
1. **圖表主題與目的**：想要傳達什麼核心觀點或數據？
2. **圖表類型選擇**：
   - **流程/關係圖**（如架構圖、心智圖、時間軸、決策流程）
   - **數據統計圖**（如折線圖、長條圖、圓餅圖）
   - **視覺概念圖**（如文宣插圖、海報、卡片式排版）
3. **輸出格式偏好**：Mermaid 語法、向量 SVG、或實體圖片檔案（PNG/JPG/WebP）。

### STEP 2：技術路徑匹配與生成

#### 路線 A：Mermaid 語法（適用於結構化圖表）
- 適用：流程圖、時序圖、心智圖、甘特圖。
- 作法：撰寫符合標準 Markdown 的 Mermaid 程式碼塊，直接內嵌於 .md 檔案中，避免文字在圖片中模糊不清。
- *安全提醒*：括號內 label 若含特殊字元需加雙引號，避免渲染錯誤。

#### 路線 B：SVG 向量圖（適用於簡報、ICON 與扁平插圖）
- 適用：自訂圖示、簡單扁平風示意圖、質感裝飾元素。
- 作法：利用 SVG XML 程式碼直接編寫向量圖形，檔案極小且能無級縮放，適用於網頁與簡報。

#### 路線 C：內建生圖工具 `generate_image`（適用於視覺海報與合成插畫）
- 適用：高質感氛圍圖、概念海報。
- 作法：使用詳細的提示詞（包含風格、配色、尺寸、不含長句文字限制）呼叫生圖工具，並輸出至專案 `assets/` 資料夾中。

#### 路線 D：Python Matplotlib/Seaborn 繪圖（適用於精確數據分析）
- 適用：折線圖、圓餅圖、熱力圖等交付級圖表。
- 作法：撰寫 Python 腳本讀取 Excel 數據並渲染存檔。

### STEP 3：色彩與排版設計原則
- **色彩搭配**：限制色系在 3 色以內。建議使用專案既有的「鋼鐵藍 (Steel Blue)」或「莫蘭迪暖色系 (Morandi)」以維持視覺質感。
- **文字簡化**：圖表內文字應以 10 字以內的關鍵詞為主，細節說明放在 Markdown 正文中。

### STEP 4：儲存與引用
1. 靜態資源請儲存於 `raw/assets/` 或 `wiki/assets/` 下。
- 圖片引用格式：`![圖表描述](file:///absolute/path/to/assets/filename.png)`
