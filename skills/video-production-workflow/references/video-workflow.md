# 影片製作工作流參考

這份參考檔保留三類影片的關鍵差異。先分流，再出骨架，最後才進製作。

## 1. 三類影片

### 活動紀錄影片

- 目的：把已發生的活動濃縮成 60 到 180 秒的紀錄短片。
- 核心：口白、大字卡、背景音樂、過場。
- 關鍵：重情緒，不重說教。

### 教學影片

- 目的：把概念講清楚，讓觀眾看完能複述或應用。
- 核心：SOIL 教學脈絡、HyperFrames 動畫、Edge-TTS 旁白。
- 關鍵：重理解，不重花俏。

### 社群科普影片

- 目的：把知識濃縮成 2 到 3 分鐘的社群短片。
- 核心：強 Hook、多版面切換、照片佐證。
- 關鍵：重節奏，不假設觀眾會開聲音。

## 2. 共同順序

### 開工

1. 確認主題、類型、受眾、片長、平台、素材來源。
2. 類型不明時，先用目的判斷。
3. 先讀對應規則，再開始寫內容。

### 產出

1. 先寫敘事大綱。
2. 再寫段落或頁面結構。
3. 接著拆成字卡、旁白、畫面、動畫。
4. 最後才整理素材與渲染流程。

### 收工

1. 檢查類型是否正確。
2. 檢查結構是否先於畫面。
3. 檢查字卡、字幕、動畫是否各司其職。
4. 若規格不夠細，再回讀這份參考檔。

## 3. 各類型骨架

### 活動紀錄

1. Hook / 醞釀
2. Setup / 展開
3. Peak / 高光
4. Turn / 反差
5. Echo / 收尾

### 教學影片

1. 概念定位稿
2. 脈絡定位稿
3. 頁面架構表
4. 旁白與畫面拆解
5. 渲染與自檢

### 社群科普

1. Hook
2. 比喻
3. 機制
4. 反轉 / 應用
5. 結語

## 4. 風格檢查

- 活動紀錄：每張圖不要停太久，情緒曲線要清楚。
- 教學影片：每頁要有教學任務，動畫要幫助理解。
- 社群科普：前 3 秒要有 Hook，至少使用 5 種版面。
- 三類影片都要先確認使用者是否已指定類型。
