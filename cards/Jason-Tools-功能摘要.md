# Card: Jason Tools 文件工具箱

## 核心觀點
Jason Tools 是一個整合式的 PDF 與 Office 文件處理平台，主打「地端自架」、「隱私安全」與「開源透明」。它解決了將敏感文件上傳至雲端工具（如線上 PDF 轉換器）可能導致的個資外洩與合規風險。

## 摘要
- **隱私安全**：所有檔案處理皆在本地伺服器完成，不離開內網。
- **功能豐富**：包含 28 個工具，涵蓋填單用印、檔案編輯、內容擷取、格式轉換、資安處理與團隊管理。
- **企業友善**：支援 LDAP/AD 認證、角色權限控制 (RBAC)、稽核記錄與記錄轉送 (SIEM 整合)。
- **技術架構**：基於 Python 3.12，整合 OxOffice/LibreOffice 處理文書格式，支援 Linux/macOS/Windows。

## 可應用情境
- **企業內部文件流**：HR 合約用印、財務報表合併、個資去識別化。
- **高度合規需求**：需要遵循個資法 (PDPA) 或 GDPR，且不允許文件上傳至公有雲。
- **自動化工作流**：利用其 API tokens 進行文件自動化處理。
